요가동호회 수업 신청

회당 최대 14명 · 수업일 오전 9시부터 선착순 신청, 오후 12시 마감 · 마감 시 자동 대기

일반 회원 모드

격주 금요일 기준일 설정

선택한 날짜를 기준으로 2주 간격 금요일이 자동으로 수업일 후보에 표시돼요.

수동으로 세션 추가

요일 제한 없이 아무 날짜나 세션으로 추가할 수 있어요.

테스트 모드

체크하면 모든 세션의 오전 9시 잠금이 풀려요. 실제 운영 시에는 꺼두세요.

주 단위 자동 수업일 생성

켜두면 앱을 열 때마다 다가오는 월/수/격주금 수업일이 자동으로 등록돼요 (수동 추가와 삭제는 그대로 가능). 관리자가 삭제한 날짜는 자동으로 다시 생기지 않아요.

월별 참석 통계

대기로만 있었던 신청은 참석으로 집계되지 않아요. 관리자가 삭제한 수업일은 통계에서도 함께 사라져요.
